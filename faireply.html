Faireply

Dans le cadre de notre formation, nous avons conçu entièrement un site permettant de poster une question auquel les utilisateurs pourront répondre, et par la suite voter pour la meilleure réponse.

Ce site a été développer en respectant l'architecture MVC (Modèle-Vue-Controleur). Il est également protégé contre les injection SQL. Nous avons également implémenté un rôle d'administrateur pouvant gérer les éléments du site web. Par ailleurs, le site utilise le système de base de données MySQL.

Durant le développement de ce site, nous avons appliqué la méthode agile afin de garder une productivité. Nous avons revoyé le code à la fin de chacun des sprints avec nos enseignants qui jouaient le rôle de client pour lequel nous travaillons.
